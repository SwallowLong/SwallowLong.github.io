    本章主要学习了以下内容：

    1. 让设计在页面中水平居中
    2. 创建两列和三列的基本布局(基本上会两列自然就会三列了)
    3. 创建固定宽度、流式和弹性布局
    4. 创建高度相等的列
    5. CSS框架和CSS系统

    居中：这个问题so easy，margin：0 auto;如果你想的是一个固定高度的文本在窗口正中间，那就margin：auto;当然如果你设置了定位的话，请将其4个方位都设置为0，当然你设置成一个比较小的值也行，注意要相等

    position中的auto,即position的奇异的流体特性，当其中一个有固定值时，另一个则自动计算，当内容设置了宽度，而left、right都设置了值但宽度不够时，这时则看你是从左往右读，还是从右往左读，我这就以从左往右读为例，则以左边的为标准，那这时的margin是怎么运转的呢：

    1. 只设置了left、right时，宽度受限的情况下，以left为标准，否则正常
    2. 即设置了left、right又设置了margin时，宽度受限时，以margin-left为标准
    3. 即设置了left、right又设置了margin，且不是使用数值，而是使用auto时，则在left基础上居中
    4. 所以当你想要无论何时都居中时，就讲left、right设置为0即可，若是只设置了left或right则会如何呢？这以left为例，当你只设置了left时，则就会到最原始的，left+margin-left

    两列布局：设置好宽度左右浮动，overflow或clear即可，三列即在两列基础上在进行一次两列

    固定、流式和弹性布局：说白了，也是一样，只是固定宽度最简单，流式你稍微换成百分数，弹性则是换成字号比例，弹性主要针对一些喜欢放大字体的用户

    对于框架，我觉得主要针对特定的站点，而且必须使用表现性的标记，与表现、标记分开有点子不符，而且有点像早期的表格设计
