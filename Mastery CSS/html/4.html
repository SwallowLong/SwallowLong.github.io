    本章主要学习了以下内容：

    1. 固定宽度和可变宽度的圆角框
    2. 滑动门技术
    3. 多个背景图像和border-radius属性
    4. CSS投影
    5. 不透明度和RGBa
    6. 让PNG适用IE的老版本
    7. 视差滚动
    8. 图像替换

    这中间讲的圆角框是用背景图片定位实现的，这可能是由于此书是在2010年5月出版，现在我不知道公司是不是，但是网上看的时候，我觉得应该比较喜欢用border-radius的多，但是不可否认这是对思维的一种开拓，能让你死板的思维稍微活跃点，这才是重点

    滑动门技术：这里仅讲述我个人的理解，我认为就是在你的文本中设置应该很大的背景，而这个背景并不会全部在文本中显露出来，在文本的一端设置一个小的背景图片，position来定位，这样若是文本扩大了的话，小图片就会相应的随着文本移动，达到所谓滑动门的效应

    多个背景的问题自己参考CSS 3，border-radius这个本来也是应该自己参考，提一下-moz-和-webkit-的问题，就举个栗子，-moz-border-radius、-webkit-border-radius、border-radius三者同时出现，还有个border-image的问题，我倒是没去深究，大家有空可以去好好理解下

    CSS投影：现在大家应该都知道box-shadow和text-shadow，这里讲的是纯css的，主要是利用背景图片和margin的负属性，当然你用relative的负属性也是一样的效果，等会看个简单的demo就自然明白了，我个人是这个属性的强迫症患者，看起来会有点立体感

    不透明度opacity：opacity属性指的是不透明度，也就是说当opacity=0时，他就是不透明度为0，而透明度为1即100%，也就是完全透明的，而在IE中还是要注意它的老版本是不支持的所以记得加上filter: alpha(opacity=0),在IE的code中opacity的值不只是0~1，而是0~100；不过opacity使用的时候，是不分主次的即文本背景完全都一起opacity，所以有了RGBa，单单的应对于背景而文本独善其身

    PNG透明度的问题，我倒是也没弄太清楚，可能是因为还没怎么接触过吧

    图像替换：即background的url属性和position+repeat属性，一般还会加点padding
