    本章主要学习了以下内容：

    1. 创建有吸引力且可访问的数据表格
    2. 创建简单和复杂的表单布局
    3. 对各种表单元素应用样式
    4. 提供可访问的表单反馈

    表格：主要讲了表格的一些元素table、thead、tbody、tfoot、tr、caption等，以及对表格施加一些样式，使得表格更具有吸引力，其中利用4边不同深浅度的border，达到斜面和突出的效果，个人觉得是个技术活

    表单布局：fieldset以及label标签的应用等，label对于增加表单的可访问性和可用性很有用，而且通过对label设置float和width可以使表单元素对齐，这个问题原来我也是没想到的，可能确实智商低下....

    反馈这个主要就是加文本的问题，由于许多屏幕阅读器会忽略表单元素之间的文本，所以最好将其放入标签内，即在label内创建子元素，这个问题我原来在使用表单元素时，直接添加文字，结果浏览器并没有显示，也是烦了我不少的时间
